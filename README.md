# 倒计时应用 (CountdownApp)

一款功能强大、高度可定制的桌面倒计时软件。支持多项目、每日一诗、自定义背景、插件扩展、主题切换等特性，适合用于考试倒计时、纪念日、项目截止日期等多种场景。

## 功能特点

- **多项目管理**：可创建多个倒计时窗口，每个窗口独立设置目标日期、时间、颜色、字体等。
- **精确倒计时**：支持到分钟级别的倒计时，自动计算剩余天数、小时、分钟以及工作日（基于中国节假日）。
- **自定义外观**：可分别设置背景颜色、背景图片、字体颜色、透明度，并支持拖拽调整窗口位置和大小。
- **每日一诗**：内置小学、初中、高中诗词库，每日随机显示一句诗词，可关闭。
- **插件系统**：支持第三方插件扩展功能，插件可访问安全API，具备权限管理、风险分析、行为监控等安全机制。
- **主题切换**：内置浅色/深色/系统原生主题，支持自定义主题并保存。
- **多语言**：支持中文和英文界面，可动态切换。
- **系统托盘**：最小化到托盘，支持右键菜单控制窗口显示/隐藏。
- **开机自启**：支持设置开机自动启动。
- **全屏模式**：可一键全屏显示倒计时，适合用作大屏展示。
- **自动字体**：窗口大小变化时自动调整字体大小。
- **配置持久化**：所有设置自动保存，下次启动恢复。
- **调试与监控**：内置日志系统、调试等级控制、状态监控，方便开发者排查问题。

## 安装与运行

### 环境要求
- Python 3.6+
- 操作系统：Windows（推荐）、Linux、macOS（部分功能如圆角、开机自启可能受限）

### 依赖库
主要依赖：
- `tkinter`（Python 标准库）
- `Pillow`（图像处理）
- `chinese_calendar`（中国节假日计算）
- `pystray`（系统托盘，可选）

安装依赖：
```bash
pip install pillow chinese-calendar pystray
```

### 运行方式
1. 下载源代码或直接运行 `djs.py`。
2. 如需打包为独立 exe，可使用 PyInstaller：
   ```bash
   pip install pyinstaller
   pyinstaller -F -w djs.py
   ```

## 使用方法

### 首次启动
启动后会在屏幕中央显示一个示例倒计时窗口。右键单击窗口可打开菜单，进行置顶、全屏、关闭等操作。左键拖拽可移动窗口，右下角拖拽可调整大小。

### 设置界面
双击托盘图标或右键点击窗口选择“设置”打开主设置窗口。设置界面包含以下选项卡：

- **项目管理**：添加、编辑、删除倒计时项目。每个项目可独立设置名称、目标日期、精确时间、是否同时显示天数与时分、背景类型（颜色/图片）、字体颜色、透明度等。
- **全局设置**：修改全局字体、语言、开机自启、每日一诗等级、查看软件信息等。
- **高级**：插件管理（加载、启用/禁用、权限设置、插件日志）、插件开发文档。
- **主题**：切换内置主题或自定义新主题，调整窗口圆角、亮度、饱和度。
- **帮助**：常见问题解答、运行状态监控、开发者工具（调试等级、查看日志、打开数据目录）。

### 自定义外观
在项目管理中点击“自定义外观”按钮，可进入可视化编辑器，拖拽调整项目名称、倒计时、诗词的位置、字体大小和颜色，并保存为自定义布局。

## 插件开发

插件系统允许开发者扩展应用功能。插件应继承 `Plugin` 类并实现相应的方法。

### 插件文件位置
将 `.py` 插件文件放入 `plugins` 文件夹，程序启动时会自动加载。

### 插件基本结构
```python
# 插件元数据（可选）
__plugin_name__ = "我的插件"
__version__ = "1.0"
__author__ = "作者"

class MyPlugin(Plugin):
    def on_load(self):
        self.api.log("插件已加载")

    def on_enable(self):
        # 插件启用时调用
        pass

    def on_disable(self):
        # 插件禁用时调用
        pass

    def on_settings_open(self, parent):
        # 在“高级”选项卡中添加自定义设置界面，返回一个 Frame
        frame = tk.Frame(parent)
        tk.Label(frame, text="插件设置").pack()
        return frame

    # 其他可用事件：on_window_create, on_window_closed, on_tick, on_before_save_config, on_after_save_config, on_settings_closed, on_poem_update, on_language_change
```

### 安全API
插件通过 `self.api` 执行受限操作，需要相应权限（由用户授予）：
- `api.log(message, level)`：记录日志。
- `api.get_config(key=None)`：获取应用配置。
- `api.get_project(index)`：获取指定项目信息。
- `api.open_file(path, mode)`：打开文件（需要文件读写权限）。
- `api.read_file(path)`：读取文件内容。
- `api.write_file(path, content)`：写入文件。
- `api.socket_connect(host, port)`：建立网络连接（需要网络权限）。
- `api.run_command(cmd)`：执行系统命令（需要命令权限）。

### 权限与安全
- 插件加载时会进行静态代码分析，识别危险导入和调用，标记风险等级（0-2）。
- 高风险插件默认启用限制模式，限制文件、网络、命令操作。
- 用户可在“高级”选项卡中为每个插件单独授予权限或启用/禁用。
- 插件运行异常时会被监控并提示用户禁用。

### 示例插件
`plugins/example_plugin.py` 提供了完整的示例代码，展示了所有可用 API 和事件。

## 主题定制

除了内置的浅色/深色/系统原生主题，用户可以自定义主题：
1. 在“主题”选项卡中调整背景色、框架色、强调色、文字色、边框色、字体、字号、圆角、亮度、饱和度。
2. 点击“保存为新主题”并输入名称，即可将当前设置保存为自定义主题。
3. 自定义主题会出现在主题下拉列表中，可随时切换或删除。

主题数据保存在 `data/countdown_config.json` 中。

## 配置与文件说明

程序运行后会创建以下目录和文件：

- `data/`：数据目录
  - `countdown_config.json`：主配置文件，包含所有项目、全局设置、插件状态、主题等。
  - `LOG.txt`：运行日志，调试时可查看。
  - `poems.json`：诗词库（小学/初中/高中）。
  - `languages.json`：多语言翻译文件。
  - `user/`：用户上传的背景图片存放目录。
  - `plugins/`：插件存放目录。
  - `debug_command.txt`：调试命令文件（由外部监控程序写入，用于动态控制应用）。

## 常见问题

**Q：为什么每次重启需要重新设置？**  
A：可能是权限不足，尝试以管理员身份运行，或检查软件工作目录（设置 → 全局设置 → 软件信息）是否正确。

**Q：设置开机自启动失败？**  
A：程序会尝试写入注册表，需要管理员权限。也可手动将快捷方式放入启动文件夹。

**Q：背景图片不显示？**  
A：图片必须放在 `data/user/` 目录下，支持格式：png、jpg、jpeg、gif、bmp。

**Q：插件加载失败？**  
A：查看 `data/LOG.txt` 获取详细错误。可能是插件代码有语法错误或依赖缺失。

**Q：如何完全重置程序？**  
A：退出程序，删除 `data/countdown_config.json`，重启后将生成默认配置。

更多问题请查看帮助选项卡中的“基础使用”和“进阶使用”。
